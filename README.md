# MuMood — Music Review & Discovery Platform

แพลตฟอร์มสำหรับค้นหาและรีวิวเพลง เชื่อมต่อ Spotify API ให้ผู้ใช้บันทึกความรู้สึกต่อเพลงในหลายมิติและจัดการเพลงโปรดในที่เดียว

---

## Features

### 1. Authentication

หน้า Login / Register มีแถบสลับโหมดด้านบน รองรับการกรอก Username หรือ Email พร้อม Password ที่ซ่อนตัวอักษร มีตัวเลือก Remember me และช่อง Confirm Password ในโหมดสมัครสมาชิก

---

### 2. Home & Discovery

แถบค้นหาสำหรับชื่อเพลง ศิลปิน หรืออัลบั้ม ดึงข้อมูลจาก Spotify API กดปกเพลงเพื่อเข้าหน้ารีวิวได้ทันที

---

### 3. Review

แสดงปกขนาดใหญ่พร้อมชื่อเพลงและศิลปิน

**Multi Rating** — Slider แยก 3 มิติ ได้แก่ Beat (จังหวะดนตรี), Lyric (เนื้อเพลง) และ Mood (อารมณ์ร่วม)

**Emotion Selector** — เลือก Emotion tag ที่สื่อถึงความรู้สึกที่มีต่อเพลง เช่น Happy, Sad, Angry, Chill เป็นต้น

**Mood Color Palette** — เลือกสีอารมณ์ของเพลง สีที่เลือกจะเปลี่ยน Background ของหน้าแบบ real-time

แสดงรีวิวของผู้ใช้อื่นพร้อมปุ่มเขียนรีวิวของตัวเอง

---

### 4. History

รายการรีวิวทั้งหมดในรูปแบบ List Header แสดงข้อมูลผู้ใช้และจำนวนรีวิวรวม แต่ละการ์ดประกอบด้วย Thumbnail ชื่อเพลง ศิลปิน คะแนน Beat/Lyric/Mood Emotion ที่เลือก และข้อความรีวิว

---

### 5. Favorite Songs

Gallery รูปแบบ Grid เน้นปกอัลบั้ม แต่ละรายการแสดงปก ชื่อเพลง ไอคอนหัวใจ และชื่อศิลปิน

---

### 6. User Profile

แสดง Avatar, Username และ Email พร้อมสถิติภาพรวม ได้แก่ จำนวน Reaction ที่ได้รับ, จำนวนรีวิวทั้งหมด และวันที่สร้างบัญชี มีปุ่ม Log out

---

## Admin

### 7. Home Admin

Toggle Menu กรองเพลงใน 3 โหมด ได้แก่ ทั้งหมด, จาก Spotify และที่แอดมินเพิ่มเอง พร้อมปุ่มเพิ่มเพลงและ Music list แสดงรายการทั้งหมด

---

### 8. Add Song

ฟอร์มสำหรับเพิ่มเพลง Custom ประกอบด้วยช่องอัปโหลดปก, ชื่อเพลง, ประเภทเพลง (Selector), ชื่อศิลปิน และปุ่ม Create สีเขียวสำหรับบันทึกข้อมูล
